Allow students to sign up themselves instead of relying on admin to create accounts. Auto-generate enrollment numbers and send notification emails to both the user and admin.

## Auto-Generation Logic

**Enrollment Number Format:** `ENR-2025-00001`

**Implementation Options:**

### Option 1: Counter-based (Recommended)
1. Count existing users with `role: 'user'`
2. Increment by 1
3. Format as: `ENR-${year}-${counter.toString().padStart(5, '0')}`

### Option 2: Timestamp-based
1. Use current timestamp: `Date.now()`
2. Format as: `ENR-${year}-${timestamp.toString().slice(-5)}`

**We'll use Option 1 for clean, sequential numbers.**

## Verification Plan

### Manual Testing
1. **Go to Login Page → Click "Sign Up"**
2. **Fill Form:** Name, Email, Password
3. **Submit**
4. **Verify:**
   - Success message shows enrollment number
   - User receives welcome email
   - Admin receives notification email
   - User can log in with enrollment number + password
   - User role is "student"
   - User can enroll in courses

### Edge Cases to Test
- Duplicate email addresses
- Weak passwords
- Invalid email formats
- Network errors during registration
